# Dexplo - A suite of data analysis libraries for Python

## Libraries

* [Dexplo](https://www.dexplo.org/dexplo)
* [Dexplot](https://www.dexplo.org/dexplot)
* [Bar Chart Race](https://www.dexplo.org/bar_chart_race)
* [DataFrame Image](https://www.dexplo.org/dataframe_image)
* [Jupyter to Medium](https://www.dexplo.org/jupyter_to_medium)
* [Minimally Sufficient Pandas](https://www.dexplo.org/minimally_sufficient_pandas)

## Overview

The overall aim of the dexplo suite of libraries is supply a powerful and efficient set of tools for doing data analysis and visualization in Python.
